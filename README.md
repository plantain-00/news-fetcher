# tools and global npm packages

+ node.js >=4
+ typescript
+ gulp
+ tsd

# development

+ `npm install`
+ `tsd install`
+ `tsc && gulp tslint`
+ `node news.js`
+ add environment variable NEWS_FETCHER_KEY
